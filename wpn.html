Waypoint Planning Networks

Waypoint Planning Networks, or WPN, is a hybrid motion planning algorithm based on LSTMs with a local kernel, a classic algorithm such as A*, and a global kernel using a learned algorithm. WPN produces a more computationally efficient and robust solution than other learned approaches. WPN is open source, see the GitHub repo for more a more technical tutorial.
